# 독해 지문 분석기 (Passage Analyzer)

영어 독해 지문을 붙여 넣으면 Claude AI가 분석해 주는 **개인용 로컬 도구**입니다.
기존 단어시험 관리 앱(Vocapassport)과는 완전히 별개로 동작합니다.

## 무엇을 해 주나요?

| 탭 | 내용 |
|---|---|
| 📌 출처 | 수능/평가원/교육청/EBS/원문(기사·서적) 어디서 온 지문인지 추정 (웹 검색으로 확인 가능) |
| 📖 지문 분석 | 주제 · 요지 · 문단별 전개 구조 · 난이도 · 전문 해석 |
| 🔤 단어 클러스터 | 핵심 단어 8~15개의 뜻, 유의어/반의어 묶음, 콜로케이션, 지문 속 예문 |
| 🧑‍🏫 수업 포인트 | 수업에서 짚어야 할 구문·문법·배경지식 |
| ✏️ 풀이 요령 | 이 지문으로 나올 만한 문제 유형별 접근법 |

분석 결과는 자동 저장되어 왼쪽 목록에서 언제든 다시 볼 수 있습니다.

## 처음 한 번만: API 키 준비

1. https://console.anthropic.com 에 가입하고 결제 수단을 등록합니다.
2. **API Keys** 메뉴에서 키를 발급받습니다 (`sk-ant-...` 로 시작).
3. 프로그램을 켠 뒤 ⚙️ 설정에 키를 붙여 넣고 저장합니다.
   (키는 이 폴더의 `data/config.json` 에만 저장되고 git에 올라가지 않습니다.)

## 실행 방법

**윈도우:** `start.bat` 더블클릭 → 브라우저가 자동으로 열립니다.

**맥/리눅스/터미널:**

```bash
cd passage-analyzer
npm install        # 처음 한 번
node server.mjs    # http://localhost:3456
```

## 비용 안내

사용한 만큼만 과금됩니다 (Anthropic API 크레딧 차감).

| 모델 | 지문 1건 분석 비용(대략) | 특징 |
|---|---|---|
| Claude Opus 4.8 (기본) | 약 $0.1 ~ 0.3 | 최고 품질 |
| Claude Sonnet 5 | 약 $0.03 ~ 0.1 | 빠르고 저렴 |

"웹 검색으로 출처 찾기"를 켜면 검색 1회당 약 $0.01이 추가됩니다 (지문당 최대 5회).
모델은 ⚙️ 설정에서 바꿀 수 있습니다.

## 동작 확인 (처음 설치 후)

1. 서버를 켜고 http://localhost:3456 접속 → 설정 창이 뜨면 API 키 저장
2. 수능 기출 지문 하나를 붙여 넣고 [분석 시작]
3. 1~3분 뒤 결과 탭 5개가 채워지면 정상입니다

## 폴더 구조

```
passage-analyzer/
├── server.mjs        # 로컬 서버 (포트 3456)
├── lib/analyze.mjs   # Claude API 호출 + 프롬프트
├── lib/store.mjs     # 분석 결과/설정 저장 (data/*.json)
├── public/           # 화면 (HTML/CSS/JS)
├── data/             # 키·분석 결과 (git 제외)
└── start.bat         # 윈도우 실행용
```
